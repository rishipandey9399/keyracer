‚òï What is Java?

üìå 1. About Java

Java is a powerful, object-oriented programming language that is:

  • Platform Independent: "Write Once, Run Anywhere" (WORA)
  • Object-Oriented: Everything is an object
  • Secure: Built-in security features
  • Robust: Strong memory management
  • Multi-threaded: Supports concurrent programming

üìå 2. Java is Used For

  • üåê Web Development: Spring, JSP, Servlets
  • üì± Mobile Apps: Android development
  • üè¢ Enterprise Applications: Large-scale systems
  • üñ•Ô∏è Desktop Apps: Swing, JavaFX
  • ‚òÅÔ∏è Cloud Applications: Microservices
  • üéÆ Games: Minecraft was built in Java!

üìå 3. Why Learn Java?

  • üíº High Demand: One of the most popular languages
  • üí∞ Great Salary: Well-paying career opportunities
  • üåç Large Community: Huge support and resources
  • üìö Rich Libraries: Extensive built-in functionality
  • üîÑ Regular Updates: Constantly evolving

üìå 4. Java vs Other Languages

Feature Java Python C++
Learning Curve Moderate Easy Hard
Performance Fast Slower Very Fast
Platform Cross-platform Cross-platform Platform-specific

üìå 5. Java Versions

  • Java 8 (LTS): Lambda expressions, Stream API
  • Java 11 (LTS): Local variable type inference
  • Java 17 (LTS): Records, sealed classes
  • Java 21 (LTS): Latest long-term support version

üí° LTS = Long Term Support (recommended for production)

üìå 6. Java Architecture

Java follows the "Write Once, Run Anywhere" principle:

  • üìù Source Code (.java) ‚Üí Written by developers
  • ‚öôÔ∏è Bytecode (.class) ‚Üí Compiled by javac
  • üîÑ JVM (Java Virtual Machine) ‚Üí Executes bytecode
  • üíª Any Platform ‚Üí Windows, Linux, macOS

üìå 7. Popular Java Frameworks

  • üå± Spring Boot: Enterprise applications
  • üÖ∞Ô∏è Android SDK: Mobile app development
  • üèóÔ∏è Hibernate: Database operations
  • üîß Apache Maven: Project management
  • üêò Apache Kafka: Streaming platform

‚òï Getting Started with Java

üìå 1. What You Need to Begin

  • A computer (Windows, macOS, or Linux)
  • Internet connection (for downloading JDK)
  • A text editor or IDE (like IntelliJ IDEA, Eclipse, or VS Code)
  • Java Development Kit (JDK)

üìå 2. Installing Java Development Kit (JDK)

‚úÖ Official Website:

Download JDK from: üëâ Oracle JDK or Eclipse Temurin

üì¶ Steps:

  1. Choose the latest LTS version (Java 21 recommended)
  2. Select your operating system
  3. Download and run the installer
  4. Follow the installation wizard

‚úÖ Verify Installation:

Open your terminal (CMD, PowerShell, or Bash):

java --version

and

javac --version

üìå 3. Setting up JAVA_HOME (Important!)

ü™ü Windows:

  1. Right-click "This PC" ‚Üí Properties
  2. Advanced System Settings ‚Üí Environment Variables
  3. New System Variable: JAVA_HOME
  4. Value: Path to JDK (e.g., C:\Program Files\Java\jdk-21)

üêß Linux/macOS:

export JAVA_HOME=/usr/lib/jvm/java-21-openjdk
export PATH=$JAVA_HOME/bin:$PATH

üìå 4. Your First Java Program

‚úÖ Create a file named HelloWorld.java:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java World!");
    }
}

‚úÖ Compile and Run:

# Run

üí° Important: File name must match the class name!

üìå 5. Java Development Tools

üîß IDEs (Integrated Development Environments):

  • IntelliJ IDEA: Most popular, excellent features

  • VS Code: Lightweight with Java extensions

üìù Text Editors:

üìå 6. Java File Structure

  • Java files end with .java
  • Compiled files end with .class
  • Each public class must be in its own file
  • File name must match the public class name
  • Java is case-sensitive

üìå 7. Understanding the Hello World Program

‚òï Java Syntax: The Basics

üìå 1. Java is Case Sensitive

Java is case-sensitive, meaning myVariable and MyVariable are different!

public class CaseSensitive {
    public static void main(String[] args) {
        int myVariable = 10;
        int MyVariable = 20;  // Different variable!
        
        System.out.println(myVariable);  // Prints: 10
        System.out.println(MyVariable);  // Prints: 20
    }
}

üìå 2. Comments

Used to explain code. They are ignored by the compiler.

// This is a single-line comment

/*
 * This is a 
 * multi-line comment
 */

/**
 * This is a JavaDoc comment
 * Used for documentation
 */

üìå 3. Print Statements

System.out.println("Hello World!");  // With new line
System.out.print("Hello ");           // Without new line
System.out.print("World!");           // Continues on same line

üìå 4. Variables Declaration

Java is statically typed - you must declare variable types.

üìå 5. String Formatting

// Concatenation

// printf style

// String.format
String message = String.format("Hello %s, you are %d", name, age);

üìù Java Variables

‚úÖ What is a Variable?

A variable is a container that holds data. In Java, you must specify the type of data it will hold.

int age = 25;           // Integer variable
String name = "John";   // String variable
double height = 5.9;    // Double variable
boolean isStudent = true; // Boolean variable

üîπ 1. Variable Naming Rules

‚úÖ Valid names:

  • Must start with a letter, underscore (_), or dollar sign ($)
  • Can contain letters, digits, underscores, and dollar signs
  • Cannot be a Java keyword
  • Case-sensitive

‚ùå Invalid names:

int 2name = 10;      // ‚ùå Cannot start with digit
int user-name = 10;  // ‚ùå Hyphens not allowed
int class = 10;      // ‚ùå 'class' is a keyword

‚úÖ Valid examples:

int userName = 10;
int _age = 25;
int $price = 100;
int name2 = 5;

üîπ 2. Variable Types

üî∏ Primitive Types:

byte smallNumber = 127;        // 8-bit integer
short mediumNumber = 32000;    // 16-bit integer
int number = 2000000000;       // 32-bit integer
long bigNumber = 9000000000L;  // 64-bit integer

float decimal = 3.14f;         // 32-bit decimal
double preciseDecimal = 3.14159; // 64-bit decimal

char letter = 'A';             // Single character
boolean flag = true;           // true or false

üî∏ Reference Types:

String text = "Hello World";
int[] numbers = {1, 2, 3, 4, 5};
Scanner input = new Scanner(System.in);

üîπ 3. Variable Declaration and Initialization

// Declaration only
int age;
String name;

// Declaration with initialization
int score = 95;
String city = "New York";

// Multiple declarations
int x, y, z;
int a = 1, b = 2, c = 3;

üîπ 4. Constants (final keyword)

final int MAX_SCORE = 100;
final double PI = 3.14159;
final String COMPANY_NAME = "TechCorp";

// MAX_SCORE = 200; // ‚ùå Error: cannot reassign

üîí Constants cannot be changed once assigned. Use UPPER_CASE naming convention.

üîπ 5. Type Casting

// Implicit casting (automatic)
int intValue = 100;
double doubleValue = intValue;  // int to double

// Explicit casting (manual)
double bigDecimal = 9.87;
int wholeNumber = (int) bigDecimal;  // 9 (loses decimal part)

// String to primitive
String numberText = "123";
int parsedNumber = Integer.parseInt(numberText);
double parsedDecimal = Double.parseDouble("45.67");

üîπ 6. Variable Scope

public class VariableScope {
    static int globalVar = 100;  // Class variable
    
    public static void main(String[] args) {
        int localVar = 50;       // Local variable
        
        if (true) {
            int blockVar = 25;   // Block variable
            System.out.println(globalVar); // ‚úÖ Accessible
            System.out.println(localVar);  // ‚úÖ Accessible
            System.out.println(blockVar);  // ‚úÖ Accessible
        }
        
        // System.out.println(blockVar); // ‚ùå Error: out of scope
    }
}

üîπ 7. Default Values

public class DefaultValues {
    // Instance variables have default values
    int number;        // 0
    double decimal;    // 0.0
    boolean flag;      // false
    String text;       // null
    
    public static void main(String[] args) {
        DefaultValues obj = new DefaultValues();
        System.out.println(obj.number);   // Prints: 0
        System.out.println(obj.flag);     // Prints: false
        
        // Local variables must be initialized before use
        int localVar;
        // System.out.println(localVar); // ‚ùå Error: not initialized
    }
}

üìä Java Data Types

‚úÖ Java Data Types Overview

Java has two categories of data types:

  • Primitive Types: Store simple values directly
  • Reference Types: Store references to objects

üîπ 1. Primitive Data Types

üî∏ Integer Types

byte smallInt = 127;           // 8-bit: -128 to 127
short mediumInt = 32000;       // 16-bit: -32,768 to 32,767
int regularInt = 2000000000;   // 32-bit: -2^31 to 2^31-1
long bigInt = 9000000000L;     // 64-bit: -2^63 to 2^63-1

System.out.println("byte: " + smallInt);
System.out.println("short: " + mediumInt);
System.out.println("int: " + regularInt);
System.out.println("long: " + bigInt);

üî∏ Floating-Point Types

float singlePrecision = 3.14f;      // 32-bit decimal
double doublePrecision = 3.14159;    // 64-bit decimal (more precise)

System.out.println("float: " + singlePrecision);
System.out.println("double: " + doublePrecision);

üî∏ Character Type

char letter = 'A';           // Single character
char unicode = '\u0041';     // Unicode for 'A'
char number = '7';           // Character '7', not number 7

System.out.println("char: " + letter);
System.out.println("unicode: " + unicode);

üî∏ Boolean Type

boolean isTrue = true;
boolean isFalse = false;
boolean result = (5 > 3);    // true

System.out.println("boolean: " + isTrue);
System.out.println("comparison: " + result);

üîπ 2. Reference Data Types

üî∏ String

String greeting = "Hello World";
String name = "Java";
String empty = "";
String nullString = null;

System.out.println("greeting: " + greeting);
System.out.println("length: " + greeting.length());
System.out.println("uppercase: " + greeting.toUpperCase());

üî∏ Arrays

int[] numbers = {1, 2, 3, 4, 5};
String[] names = {"Alice", "Bob", "Charlie"};
double[] prices = new double[3];  // Creates array of size 3

System.out.println("first number: " + numbers[0]);
System.out.println("array length: " + numbers.length);

üîπ 4. Type Conversion

üî∏ Automatic (Implicit) Conversion

byte b = 10;
short s = b;    // byte ‚Üí short
int i = s;      // short ‚Üí int
long l = i;     // int ‚Üí long
float f = l;    // long ‚Üí float
double d = f;   // float ‚Üí double

System.out.println("Automatic conversion: " + d);

üî∏ Manual (Explicit) Conversion

double largeNumber = 123.456;
int wholeNumber = (int) largeNumber;  // 123 (loses decimal)

long bigValue = 1000L;
int smallValue = (int) bigValue;      // Possible data loss

System.out.println("Explicit conversion: " + wholeNumber);

üîπ 5. Wrapper Classes

Each primitive type has a corresponding wrapper class:

// Primitive ‚Üí Wrapper (Boxing)
int primitive = 10;
Integer wrapper = Integer.valueOf(primitive);

// Wrapper ‚Üí Primitive (Unboxing)
Integer wrapperObj = 20;
int primitiveValue = wrapperObj.intValue();

// Autoboxing (automatic conversion)
Integer auto = 30;           // int ‚Üí Integer
int autoValue = auto;        // Integer ‚Üí int

System.out.println("Wrapper methods:");
System.out.println("Max int: " + Integer.MAX_VALUE);
System.out.println("Parse string: " + Integer.parseInt("123"));

üîπ 6. Common Type Operations

// Math operations
int a = 15, b = 4;
System.out.println("Addition: " + (a + b));        // 19
System.out.println("Division: " + (a / b));        // 3 (integer)
System.out.println("Division: " + (a / 4.0));      // 3.75 (double)
System.out.println("Modulus: " + (a % b));         // 3

// Boolean operations
boolean x = true, y = false;
System.out.println("AND: " + (x && y));            // false
System.out.println("OR: " + (x || y));             // true
System.out.println("NOT: " + (!x));                // false

// Character operations
char ch = 'A';
System.out.println("ASCII value: " + (int) ch);    // 65
System.out.println("Next char: " + (char)(ch + 1)); // B
